# Obsidian-theme-Incredible-Hulk
A custom CSS theme for Obsidian inspired by The Incredible Hulk.
Dark theme only.



## Instructions

1. In Obsidian, click Settings->Plugins and turn on "Custom CSS".
2. Go to Settings -> Community themes to activate
